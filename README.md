# Spring Boot NE Template 2025

This is a Spring Boot template project for 2025 using Java 24.

## Overview

This template provides a basic structure for building Spring Boot applications with Jakarta EE and Spring MVC.

## Technical Stack

- Java 24
- Spring Boot 3.4.5
- Jakarta EE
- Spring MVC
- Maven

## Getting Started

### Prerequisites

- JDK 24
- Maven 3.8+
